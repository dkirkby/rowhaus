RowHaus Debugger

You should see one line added below each time the seat moves forwards or backwards over the magnetic sensor. If you do not see any entries, try disconnecting the jumper wires from the rowing machine 2-pin connector and briefly touch them together to simulate the magnetic sensor. The data fields are:

  • count: The number of sensor pulses since the server on the RPi started. This should increment by one each time.
  • dt1: the duration of the sensor pulse in milliseconds, which measures how fast the seat is moving when it passes the sensor.
  • dt2: the elapsed time since the last sensor pulse in milliseconds, which measures the stroke rate.
  • t: the server unix timestamp for this sensor pulse.
Values are rounded to three decimals in the printout below.
